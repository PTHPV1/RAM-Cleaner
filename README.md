# RAM Cleaner

โปรเจกต์ `RAM Cleaner` เป็นแอป Windows Forms สำหรับช่วยดูสถานะหน่วยความจำและสั่งทำความสะอาด RAM บน Windows แบบใช้งานง่าย

## คำอธิบายโปรเจกต์

`RAM Cleaner` ถูกพัฒนาขึ้นเพื่อเป็นเครื่องมือเดสก์ท็อปสำหรับผู้ใช้ Windows ที่ต้องการตรวจสอบสถานะการใช้หน่วยความจำของเครื่องและสั่งคืนหน่วยความจำบางส่วนให้ระบบแบบรวดเร็วผ่านหน้าจอที่ใช้งานง่าย โปรแกรมจะแสดงข้อมูล RAM แบบเรียลไทม์ พร้อมชุดคำสั่งสำหรับลดการใช้หน่วยความจำของโปรเซสและล้างหน่วยความจำแคชของระบบในส่วน `Standby List`

เป้าหมายของโปรเจกต์นี้คือช่วยให้ผู้ใช้มองเห็นภาพรวมของหน่วยความจำได้ชัดขึ้น และสามารถสั่งทำความสะอาด RAM ได้จากปุ่มควบคุมเพียงไม่กี่จุด โดยไม่ต้องใช้เครื่องมือระดับระบบที่ซับซ้อนโดยตรง ทั้งนี้โปรแกรมไม่ได้เพิ่ม RAM จริง แต่ช่วยให้ Windows จัดสรรหน่วยความจำใหม่ได้สะดวกขึ้นในบางสถานการณ์

## ความสามารถหลัก

- แสดงปริมาณ RAM ที่ใช้อยู่ แบบว่าง และทั้งหมดแบบเรียลไทม์
- แสดงสถานะการใช้หน่วยความจำเป็นเปอร์เซ็นต์
- ลด `Working Set` ของโปรเซสที่กำลังทำงาน
- ล้าง `Standby List` เพื่อคืนหน่วยความจำแคชให้ระบบ
- แสดงบันทึกการทำงานภายในโปรแกรม
- รองรับการเปิดใหม่แบบผู้ดูแลระบบสำหรับคำสั่งที่ต้องใช้สิทธิ์เพิ่มเติม

## เทคโนโลยีที่ใช้

- C#
- Windows Forms
- .NET Framework 4.8
- Windows Native API (`kernel32`, `psapi`, `advapi32`, `ntdll`)

## โครงสร้างไฟล์สำคัญ

- `Form1.cs` จัดการ UI, event และ flow การสั่งงาน
- `Form1.Designer.cs` จัดวางหน้าจอและคอนโทรล
- `MemoryStatusProvider.cs` อ่านข้อมูลหน่วยความจำจาก Windows
- `RamCleanerService.cs` รวมคำสั่งลด Working Set และล้าง Standby List

## วิธีเปิดโปรเจกต์

1. เปิดโปรเจกต์ด้วย Visual Studio
2. โหลดไฟล์ `RAM_Cleaner.csproj`
3. Build แบบ `Debug` หรือ `Release`
4. รันโปรแกรมจาก Visual Studio หรือไฟล์ `bin/Debug/RAM_Cleaner.exe`

## วิธีใช้งาน

1. เปิดโปรแกรม
2. ดูสถานะ RAM จากหน้าแดชบอร์ด
3. กด `ทำความสะอาดอัตโนมัติ` เพื่อสั่งทำงานทั้งสองขั้นตอน
4. หากต้องการควบคุมเอง ให้ใช้ปุ่ม `ลด Working Set` หรือ `ล้าง Standby List`
5. ถ้าระบบแจ้งเรื่องสิทธิ์ ให้กด `เปิดใหม่แบบผู้ดูแลระบบ`

## ข้อควรระวัง

- การล้าง RAM มีผลเชิงชั่วคราว ระบบและโปรแกรมต่าง ๆ อาจดึงหน่วยความจำกลับมาใช้อีกได้
- การล้าง `Standby List` อาจทำให้บางโปรแกรมหรือเกมโหลดข้อมูลใหม่และหน่วงชั่วคราว
- คำสั่งบางอย่างต้องใช้สิทธิ์ผู้ดูแลระบบ
- โปรแกรมนี้ไม่ได้เพิ่ม RAM จริง แต่ช่วยคืนหน่วยความจำบางส่วนให้ระบบจัดสรรใหม่

## สถานะโปรเจกต์

โปรเจกต์นี้อยู่ในรูปแบบเดสก์ท็อปแอปสำหรับ Windows และเหมาะสำหรับต่อยอดเพิ่มเติม เช่น

- ตั้งเวลาเคลียร์ RAM อัตโนมัติ
- เคลียร์เฉพาะโปรเซสที่เลือก
- บันทึกประวัติการทำงานลงไฟล์
- เพิ่มการตั้งค่าและหน้าจอภาษาไทยเต็มรูปแบบ
